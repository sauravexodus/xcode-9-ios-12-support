# Xcode 9 iOS 12 Support

Support files to deploy iOS apps on devices with iOS 12 and newer versions.

## Steps to install
- `git clone git@github.com:sauravexodus/xcode-9-ios-12-support.git`
- `cd xcode-9-ios-12-support`
- `. ./install.sh`
- Restart Xcode
